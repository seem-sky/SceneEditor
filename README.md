这是一个开发中的简单的场景编辑器，提供简单的精灵摆放以及多边形编辑。

开发环境：win7、vs2013、qt5.3、cocos2dx2.2.2

PS：编辑器由qt5.3和略加修改过的cocos2dx2.2混合编写，由于cocos2dx3.0使用了glfw使得glfw的消息机制和qt的消息机制可能发生冲突，至少是在mac环境下会导致程序崩溃，最后不得已使用了较老版本的cocos2dx2.2,而2.x的消息机制是分平台独立编写,使得qt和cocos2dx消息传递的处理也依赖于相应平台。而此版本在vs2013环境下开发，所以也只适用于windows，若能针对平台消息传递机制修改相应代码应该是能够实现跨平台。
